# Alishagram 📸

Alishagram is a premium, high-end editorial photo-sharing platform built with a **Brutalist Minimalism** aesthetic. It focuses on large-scale typography, cinematic transitions, and a tactile, physical magazine feel.

## ✨ Features

- **Brutalist Editorial UI**: A high-contrast dark theme with sharp corners and bold 'Anton' typography.
- **Cinematic Experience**: Page transitions featuring blur and scale effects for a premium feel.
- **Tactile Film Grain**: A global animated noise overlay that gives the site a printed, physical texture.
- **Masonry Story Feed**: A staggering vertical photo grid that creates a dynamic, magazine-style layout.
- **Editorial Hover Reveals**: Photos default to greyscale and reveal their vivid colors on hover.
- **Split-Screen Authentication**: A dramatic opening experience for users logging in or registering.
- **Dark Mode Map Integration**: Leaflet maps synchronized with the moody, dark-mode styling.
- **Custom System Cursor**: A stark, interactive brutalist cursor that adapts to elements.
- **Real-time Progress Tracking**: A sleek editorial scroll bar for visual feedback.

## 🚀 Tech Stack

- **Frontend**: React 19, Vite, Framer Motion, Lucide React, Leaflet.
- **Backend**: Node.js, Express, MongoDB, Cloudinary (for image hosting).
- **Styling**: Vanilla CSS with a heavy focus on custom variables and layout layering.

## 🛠️ Getting Started

### Prerequisites
- Node.js & npm
- MongoDB Atlas account
- Cloudinary account

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Faizanhassan47/Alishagram.git
   ```
2. Setup Backend:
   - Navigate to `/backend`
   - Create a `.env` file with `MONGODB_URI`, `JWT_SECRET`, `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, and `CLOUDINARY_API_SECRET`.
   - Run `npm install`
   - Run `npm run dev`

3. Setup Frontend:
   - Navigate to `/frontend`
   - Run `npm install`
   - Run `npm run dev`
